Introduction

RB BASIC (Ronen Blumberg BASIC) is a modern BASIC compiler that provides full compatibility with GW-BASIC and QBasic syntax while compiling to native executable code through C. It combines the simplicity of classic BASIC with modern performance.

Key Features

  • Full GW-BASIC/QBasic compatibility
  • Compiles to native Windows executables
  • Line numbers optional
  • Structured programming with SUB and FUNCTION
  • Graphics support via SDL2
  • File I/O capabilities
  • Arrays and DATA statements
  • String manipulation functions

Program Basics

Line Numbers

RB BASIC supports traditional line numbers for compatibility with classic BASIC programs:

10 PRINT "Hello, World!"
20 END

Comments

Use REM to add comments to your code:

10 REM This is a comment
20 PRINT "Hello"  REM Comments can also go at the end of lines

Multiple Statements Per Line

Use colons (:) to separate multiple statements on one line:

10 A = 5: B = 10: PRINT A + B

Variables and Data Types

Numeric Variables

Numeric variables can store integers or floating-point numbers:

X = 10
Y = 3.14159
TOTAL = X * Y

String Variables

String variables end with a dollar sign ($):

NAME$ = "Ronen"
MESSAGE$ = "Hello, " + NAME$
PRINT MESSAGE$
Note: Variable names are case-insensitive. name$, NAME$, and Name$ all refer to the same variable.

LET Statement

The LET keyword is optional for assignments:

LET X = 100  REM Optional LET
Y = 200         REM Without LET

Operators

Arithmetic Operators

Operator Description Example
+ Addition A = 5 + 3
- Subtraction B = 10 - 4
* Multiplication C = 6 * 7
/ Division D = 20 / 4
\ Integer Division E = 17 \ 5 (= 3)
MOD Modulo (Remainder) F = 17 MOD 5 (= 2)
^ Exponentiation G = 2 ^ 8 (= 256)

Comparison Operators

Operator Description Example
= Equal to IF A = 5 THEN
<> Not equal to IF B <> 0 THEN
< Less than IF X < 100 THEN
> Greater than IF Y > 50 THEN
<= Less than or equal IF A <= 10 THEN
>= Greater than or equal IF B >= 5 THEN

Logical Operators

Operator Description Example
AND Logical AND IF A > 0 AND B < 100 THEN
OR Logical OR IF X = 1 OR X = 2 THEN
NOT Logical NOT IF NOT EOF(1) THEN

Control Flow

IF...THEN...ELSE

Single-line IF

IF condition THEN statement 
IF A > 10 THEN PRINT "Greater"

Block IF (Multi-line)

IF condition THEN
    statements
ELSEIF condition THEN
    statements
ELSE
    statements
END IF 
IF SCORE >= 90 THEN
    PRINT "Grade: A"
ELSEIF SCORE >= 80 THEN
    PRINT "Grade: B"
ELSEIF SCORE >= 70 THEN
    PRINT "Grade: C"
ELSE
    PRINT "Grade: F"
END IF

FOR...NEXT Loop

FOR variable = start TO end [STEP increment]
    statements
NEXT [variable] 
REM Count from 1 to 10
FOR I = 1 TO 10
    PRINT I
NEXT I

REM Count by 2s
FOR J = 0 TO 20 STEP 2
    PRINT J
NEXT J

REM Count backwards
FOR K = 10 TO 1 STEP -1
    PRINT K
NEXT K

WHILE...WEND Loop

WHILE condition
    statements
WEND 
I = 1
WHILE I <= 10
    PRINT I
    I = I + 1
WEND

DO...LOOP

DO WHILE condition
    statements
LOOP 
REM DO WHILE (test at start)
I = 1
DO WHILE I <= 5
    PRINT I
    I = I + 1
LOOP

REM DO...LOOP WHILE (test at end)
J = 1
DO
    PRINT J
    J = J + 1
LOOP WHILE J <= 5

REM DO UNTIL
K = 1
DO UNTIL K > 5
    PRINT K
    K = K + 1
LOOP

GOTO Statement

GOTO line_number 
10 PRINT "Start"
20 GOTO 40
30 PRINT "Skipped"
40 PRINT "End"

GOSUB...RETURN

GOSUB line_number
...
line_number: statements
RETURN 
10 GOSUB 100
20 GOSUB 100
30 END
100 PRINT "Subroutine called"
110 RETURN

ON...GOTO / ON...GOSUB

ON expression GOTO line1, line2, line3...
ON expression GOSUB line1, line2, line3... 
INPUT "Choose 1-3: ", CHOICE
ON CHOICE GOTO 100, 200, 300
100 PRINT "Option 1": END
200 PRINT "Option 2": END
300 PRINT "Option 3": END

END Statement

Terminates program execution:

END

Input/Output

PRINT Statement

PRINT [expression [; | ,] expression...] 
PRINT "Hello, World!"
PRINT "X ="; X
PRINT A; B; C              REM Semicolon: no spacing
PRINT A, B, C              REM Comma: tab spacing
PRINT A;                   REM No newline at end

INPUT Statement

INPUT ["prompt";] variable [, variable...] 
INPUT "Enter your name: ", NAME$
INPUT "Enter age: ", AGE
INPUT "X, Y: ", X, Y

LINE INPUT Statement

Reads an entire line including commas and quotes:

LINE INPUT "Enter a sentence: ", TEXT$

CLS (Clear Screen)

CLS

LOCATE

LOCATE row, column 
LOCATE 10, 20
PRINT "At position 10,20"

COLOR

COLOR foreground [, background] 
COLOR 15, 1  REM White on blue

KEY Statement

KEY OFF  REM Hide function key line
KEY ON   REM Show function key line

DATA Statements

DATA

Stores data values to be read by READ statements:

DATA value1, value2, value3...

READ

Reads values from DATA statements:

READ variable [, variable...]

RESTORE

Resets the data pointer to the beginning or a specific line:

RESTORE [line_number]
Example: 
10 DATA "Alice", 25, "Bob", 30
20 DATA "Charlie", 35
30 READ NAME$, AGE
40 PRINT NAME$; " is "; AGE
50 READ NAME$, AGE
60 PRINT NAME$; " is "; AGE
70 RESTORE
80 READ NAME$, AGE
90 PRINT "First again: "; NAME$

Built-in Functions

Mathematical Functions

Function Description Example
ABS(x) Absolute value Y = ABS(-5) → 5
SIN(x) Sine (radians) Y = SIN(1.57)
COS(x) Cosine (radians) Y = COS(0) → 1
TAN(x) Tangent (radians) Y = TAN(0.785)
ATN(x) Arctangent (radians) Y = ATN(1) → 0.785
SQR(x) Square root Y = SQR(16) → 4
EXP(x) e to the power of x Y = EXP(1) → 2.718
LOG(x) Natural logarithm Y = LOG(10)
INT(x) Integer part (floor) Y = INT(3.7) → 3
FIX(x) Truncate to integer Y = FIX(-3.7) → -3
CINT(x) Round to nearest integer Y = CINT(3.5) → 4
SGN(x) Sign (-1, 0, or 1) Y = SGN(-5) → -1
RND Random number 0 to 1 Y = RND

String Functions

Function Description Example
LEFT$(s$, n) Leftmost n characters A$ = LEFT$("HELLO", 2) → "HE"
RIGHT$(s$, n) Rightmost n characters A$ = RIGHT$("HELLO", 2) → "LO"
MID$(s$, start, len) Substring A$ = MID$("HELLO", 2, 3) → "ELL"
LEN(s$) Length of string N = LEN("HELLO") → 5
INSTR(s1$, s2$) Find substring position N = INSTR("HELLO", "LL") → 3
CHR$(n) Character from ASCII code A$ = CHR$(65) → "A"
ASC(s$) ASCII code of first char N = ASC("A") → 65
STR$(n) Number to string A$ = STR$(123) → " 123"
VAL(s$) String to number N = VAL("123") → 123
UCASE$(s$) Convert to uppercase A$ = UCASE$("hello") → "HELLO"
LCASE$(s$) Convert to lowercase A$ = LCASE$("HELLO") → "hello"
SPACE$(n) String of n spaces A$ = SPACE$(5)
STRING$(n, c$) Repeat character n times A$ = STRING$(5, "*") → "*****"
HEX$(n) Number to hexadecimal A$ = HEX$(255) → "FF"
OCT$(n) Number to octal A$ = OCT$(8) → "10"

Input Functions

Function Description Example
INKEY$ Get keystroke (no wait) K$ = INKEY$
INPUT$(n) Read n characters A$ = INPUT$(5)

System Functions

Function Description Example
TIMER Seconds since midnight T = TIMER
DATE$ Current date D$ = DATE$ → "10-30-2025"
TIME$ Current time T$ = TIME$ → "14:30:25"

RANDOMIZE

Initializes random number generator:

RANDOMIZE TIMER  REM Seed with current time

User-Defined Functions

DEF FNname(parameters) = expression 
DEF FNsquare(X) = X * X
DEF FNarea(L, W) = L * W

PRINT FNsquare(5)     REM Prints 25
PRINT FNarea(10, 20) REM Prints 200

Arrays

DIM Statement

Declares array dimensions:

DIM array_name(size [, size2...]) 
REM One-dimensional array
DIM SCORES(10)
FOR I = 0 TO 10
    SCORES(I) = I * 10
NEXT I

REM Two-dimensional array
DIM GRID(5, 5)
FOR X = 0 TO 5
    FOR Y = 0 TO 5
        GRID(X, Y) = X + Y
    NEXT Y
NEXT X

REM String arrays
DIM NAMES$(100)
NAMES$(0) = "Alice"
NAMES$(1) = "Bob"

LBOUND and UBOUND

Get lower and upper bounds of arrays:

DIM A(10)
PRINT LBOUND(A)  REM Prints 0
PRINT UBOUND(A)  REM Prints 10

Subroutines and Functions

SUB Procedures

SUB name(parameters)
    statements
END SUB 
SUB DrawBox(X, Y, W, H)
    LINE (X, Y)-(X+W, Y)
    LINE (X+W, Y)-(X+W, Y+H)
    LINE (X+W, Y+H)-(X, Y+H)
    LINE (X, Y+H)-(X, Y)
END SUB

REM Call the SUB
CALL DrawBox(100, 100, 50, 50)

FUNCTION Procedures

FUNCTION name(parameters)
    statements
    name = return_value
END FUNCTION 
FUNCTION Factorial(N)
    IF N <= 1 THEN
        Factorial = 1
    ELSE
        Factorial = N * Factorial(N - 1)
    END IF
END FUNCTION

PRINT Factorial(5)  REM Prints 120

File Input/Output

OPEN Statement

OPEN filename$ FOR mode AS #filenumber

Modes: INPUT, OUTPUT, APPEND

CLOSE Statement

CLOSE [#filenumber [, #filenumber...]]

Writing to Files

REM Write to a file
OPEN "DATA.TXT" FOR OUTPUT AS #1
PRINT #1, "Line 1"
PRINT #1, "Line 2"
CLOSE #1

Reading from Files

REM Read from a file
OPEN "DATA.TXT" FOR INPUT AS #1
WHILE NOT EOF(1)
    LINE INPUT #1, TEXT$
    PRINT TEXT$
WEND
CLOSE #1

File Functions

Function Description
EOF(filenumber) Returns -1 if end of file reached
LOF(filenumber) Returns length of file
Complete File Example: 
REM Save and load high scores
DIM SCORES(5)
DIM NAMES$(5)

REM Save scores
OPEN "SCORES.DAT" FOR OUTPUT AS #1
FOR I = 0 TO 5
    PRINT #1, NAMES$(I)
    PRINT #1, SCORES(I)
NEXT I
CLOSE #1

REM Load scores
OPEN "SCORES.DAT" FOR INPUT AS #1
FOR I = 0 TO 5
    LINE INPUT #1, NAMES$(I)
    INPUT #1, SCORES(I)
NEXT I
CLOSE #1

Graphics Commands

Note: Graphics features require SDL2 and are rendered in a graphical window.

SCREEN

Set screen mode:

SCREEN 0  REM Text mode (80x25)
SCREEN 12 REM Graphics mode (640x480)

PSET

Plot a point:

PSET (x, y) [, color] 
PSET (100, 100), 15  REM White pixel

LINE

Draw a line or box:

LINE (x1, y1)-(x2, y2) [, color] [, B|BF] 
REM Draw a line
LINE (10, 10)-(100, 100), 14

REM Draw a box (outline)
LINE (50, 50)-(150, 100), 10, B

REM Draw a filled box
LINE (200, 50)-(300, 100), 12, BF

CIRCLE

Draw a circle or ellipse:

CIRCLE (x, y), radius [, color] [, start, end] [, aspect] 
REM Draw a circle
CIRCLE (320, 240), 50, 14

REM Draw an ellipse
CIRCLE (320, 240), 50, 12, , , 0.5

PAINT

Fill an area with color:

PAINT (x, y), color [, border_color] 
CIRCLE (320, 240), 50, 15
PAINT (320, 240), 12, 15  REM Fill the circle

POINT

Read pixel color:

color = POINT(x, y) 
C = POINT(100, 100)
PRINT "Color at 100,100 is"; C

Sound Commands

SOUND

Produce a sound of specified frequency and duration:

SOUND frequency, duration

Frequency is in Hertz (Hz), duration in clock ticks (18.2 per second)

REM Play middle C for 1 second
SOUND 262, 18.2

REM Play a scale
SOUND 262, 9  REM C
SOUND 294, 9  REM D
SOUND 330, 9  REM E
SOUND 349, 9  REM F

PLAY

Play music using Music Macro Language (MML):

PLAY music_string$ 
REM Play a simple melody
PLAY "CDEFGAB"

REM With note lengths and octaves
PLAY "O4 L4 C D E F G A B O5 C"

BEEP

Produce a system beep sound:

BEEP

SLEEP

Pause execution for specified seconds:

SLEEP 2  REM Sleep for 2 seconds

Example 4: Using SUB and FUNCTION

SUB DrawMenu()
    CLS
    PRINT "===== MAIN MENU ====="
    PRINT "1. Start Game"
    PRINT "2. High Scores"
    PRINT "3. Exit"
    PRINT
END SUB

FUNCTION GetChoice()
    INPUT "Choose (1-3): ", C
    IF C < 1 OR C > 3 THEN
        GetChoice = 0
    ELSE
        GetChoice = C
    END IF
END FUNCTION

REM Main program
DO
    CALL DrawMenu()
    CHOICE = GetChoice()
    IF CHOICE = 1 THEN PRINT "Starting game..."
    IF CHOICE = 2 THEN PRINT "Showing scores..."
LOOP UNTIL CHOICE = 3
PRINT "Goodbye!"
END
