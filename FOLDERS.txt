RB BASIC Portable Edition - Folder Structure
=============================================

This file explains what goes in each folder.

ROOT FOLDER (rbbasic-portable\)
================================
  rbbasic.exe          - Main compiler (created by build.bat)
  rbc.config           - Configuration file
  README.txt           - Quick start guide
  build.bat            - Build compiler from source
  setup.bat            - Validate installation
  clean.bat            - Clean temporary files
  compile.bat          - Generated by compiler (appears after rbbasic.exe runs)
  SDL2.dll             - Copied here automatically if graphics used


mingw32\ - YOU ADD THIS
=======================
Download from: https://winlibs.com
Choose: i686 (32-bit) version with MSVCRT runtime

Extract so you have:
  mingw32\
    bin\
      gcc.exe
      g++.exe
      as.exe
      ld.exe
      ... (other tools)
    lib\
    include\
    libexec\


lib\ - Libraries
================
This folder contains external libraries.

lib\SDL2-mingw32\ - YOU ADD THIS (OPTIONAL)
--------------------------------------------
Download from: https://github.com/libsdl-org/SDL/releases
Get: SDL2-devel-x.x.x-mingw.tar.gz

Extract and place the i686-w64-mingw32 folder here:
  lib\
    SDL2-mingw32\
      i686-w64-mingw32\
        include\
          SDL2\
            SDL.h
            ... (other headers)
        lib\
          libSDL2.a
          libSDL2main.a
          ... (other libraries)
        bin\
          SDL2.dll

NOTE: SDL2 is OPTIONAL. Only needed for graphics programs.
      Text-only programs work fine without SDL2.


src\ - Source Code
==================
Contains the RB BASIC compiler source code.
Already included:
  main.cpp             - Main compiler driver
  config.cpp/h         - Configuration system
  lexer.cpp/h          - Lexical analyzer
  parser.cpp/h         - Parser
  codegen.cpp/h        - Code generator
  token.cpp/h          - Token handling
  ast.h                - Abstract syntax tree


examples\ - Example Programs
=============================
Contains example BASIC programs.
Already included:
  hello.bas            - Hello World
  loops.bas            - Loop demonstrations
  guess.bas            - Number guessing game
  strings.bas          - String functions demo
  graphics.bas         - Graphics demo (requires SDL2)

You can add your own .BAS files here.


docs\ - Documentation
=====================
Contains manuals and guides.
Already included:
  RBBASIC_LANGUAGE_MANUAL.html    - Full manual (open in browser)
  RBBASIC_LANGUAGE_MANUAL.md      - Full manual (Markdown)
  RBBASIC_QUICK_REFERENCE.md      - Quick reference card
  PORTABLE_DISTRIBUTION_GUIDE.md  - How to create portable package
  QUICK_START_PORTABLE.md         - Quick setup guide


output\ - Temporary Files
=========================
Created automatically by setup.bat.
Used for temporary build files during compilation.
Can be cleaned with clean.bat.


SETUP CHECKLIST
===============

[ ] 1. Download and add mingw32\ folder
[ ] 2. (Optional) Download and add lib\SDL2-mingw32\
[ ] 3. Run setup.bat to validate
[ ] 4. Run build.bat to compile rbbasic.exe
[ ] 5. Test with: rbbasic examples\hello.bas

After setup, you can:
  - Copy entire folder to USB drive
  - Move to any location
  - Share with others
  - No installation or configuration needed!


WHAT YOU NEED TO ADD
====================

REQUIRED:
  mingw32\             (MinGW32 GCC compiler)

OPTIONAL:
  lib\SDL2-mingw32\    (For graphics support)


SIZE ESTIMATES
==============
  RB BASIC compiler:        ~1 MB
  Source code (src\):       ~50 KB
  Documentation (docs\):    ~100 KB
  Examples (examples\):     ~10 KB
  MinGW32 (full):           ~200-300 MB
  SDL2 (optional):          ~5-10 MB
  
  TOTAL (without MinGW):    ~1 MB
  TOTAL (with MinGW):       ~200-300 MB
  TOTAL (with SDL2):        ~210-320 MB
